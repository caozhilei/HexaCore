# 技能（Skills）层技术方案文档

## 1. 概述

### 1.1 技能层在六元组架构中的定位
技能层是六元组架构中的核心能力单元，负责实现声明式的业务逻辑封装和热插拔机制。基于HexaCore Skills平台和MCP（Model Context Protocol）协议，技能层为不同Agent提供标准化的能力调用接口，支持技能市场的生态建设。

### 1.2 设计目标
1. **声明式定义**：遵循HexaCore SKILL.md标准，实现技能元数据的结构化定义
2. **热插拔机制**：支持技能动态加载和卸载，无需重启系统
3. **权限解耦**：实现技能与权限控制的松耦合设计，基于HexaCore工具策略模型
4. **生态集成**：提供完整的Skill开发SDK和测试框架，与HexaCore ClawHub技能生态无缝集成

### 1.3 核心价值
- **业务复用**：避免相同功能在不同Agent中重复实现
- **快速集成**：第三方能力可通过标准化接口快速接入
- **安全可控**：细粒度的权限控制和安全隔离机制
- **持续进化**：技能市场支持版本管理和自动更新

## 4. 技能执行流程

### 4.1 完整执行流程
技能层的执行流程遵循标准的请求-响应模式，包含以下关键步骤：

1. **请求接收**：从频道层接收标准化的技能调用请求
2. **技能解析**：根据技能名称查找并加载对应技能
3. **权限验证**：检查调用方Agent是否具备该技能的执行权限
4. **参数验证**：验证输入参数的类型、范围和必填项
5. **沙箱执行**：在安全沙箱中执行技能逻辑
6. **结果处理**：处理执行结果，包括异常捕获和转换
7. **响应返回**：将标准化结果返回给调用方

### 4.2 核心算法实现

#### 4.2.1 技能加载与发现算法
```typescript
class SkillLoader {
  async discoverAndLoadSkills(
    skillsDir: string
  ): Promise<Map<string, SkillDefinition>> {
    const skillMap = new Map();
    
    // 1. 扫描skills目录
    const skillDirs = await fs.readdir(skillsDir, { withFileTypes: true });
    
    for (const dir of skillDirs) {
      if (dir.isDirectory()) {
        const skillPath = path.join(skillsDir, dir.name);
        
        // 2. 查找SKILL.md文件
        const skillMdPath = path.join(skillPath, 'SKILL.md');
        if (await fs.pathExists(skillMdPath)) {
          // 3. 解析技能定义
          const definition = await this.parseSkillDefinition(skillMdPath);
          
          // 4. 验证技能完整性
          if (await this.validateSkill(definition, skillPath)) {
            skillMap.set(definition.name, definition);
          }
        }
      }
    }
    
    return skillMap;
  }
}
```

#### 4.2.2 沙箱执行算法
```typescript
class SandboxExecutor {
  async executeInSandbox(
    skill: SkillDefinition,
    input: SkillInput,
    securityContext: SecurityContext
  ): Promise<SkillOutput> {
    // 1. 创建安全沙箱
    const sandbox = await this.createSandbox(skill, securityContext);
    
    try {
      // 2. 准备执行环境
      await this.prepareEnvironment(sandbox, skill, input);
      
      // 3. 应用安全策略
      await this.applySecurityPolicies(sandbox, skill.toolPermissions);
      
      // 4. 执行技能
      const result = await sandbox.execute(input);
      
      // 5. 清理和安全检查
      await this.cleanupAndAudit(sandbox, result);
      
      return result;
    } finally {
      // 6. 销毁沙箱
      await sandbox.destroy();
    }
  }
}
```

## 5. 性能优化策略

### 5.1 技能预热机制
针对高频使用的技能，系统实现预加载机制以减少启动延迟：

```typescript
class SkillWarmupManager {
  private warmupThreshold = 0.8; // 使用频率阈值
  private warmupQueue = new PriorityQueue();
  
  async scheduleWarmup(skillUsageStats: SkillUsageStats[]) {
    // 识别高频技能
    const highFrequencySkills = skillUsageStats.filter(
      stats => stats.frequency >= this.warmupThreshold
    );
    
    // 按优先级排序
    highFrequencySkills.sort((a, b) => b.priority - a.priority);
    
    // 异步预加载
    for (const skillStats of highFrequencySkills) {
      this.warmupQueue.enqueue(async () => {
        await this.preloadSkill(skillStats.skillName);
      });
    }
  }
}
```

### 5.2 结果缓存策略
基于输入参数的哈希值缓存技能执行结果，显著提高响应速度：

1. **缓存键生成**：`cacheKey = md5(skillName + JSON.stringify(inputParams))`
2. **缓存时效**：根据技能特性设置不同TTL（默认30分钟）
3. **缓存失效**：当技能版本更新或依赖数据变更时自动失效

### 5.3 并发控制机制
防止资源竞争和系统过载，实现细粒度的并发控制：

| 资源类型 | 并发限制 | 监控指标 |
|----------|----------|----------|
| CPU密集型技能 | 2个/节点 | CPU使用率 >80%时限流 |
| I/O密集型技能 | 10个/节点 | 内存使用率 >70%时告警 |
| 网络请求技能 | 5个/节点 | 网络延迟 >100ms时降级 |

### 5.4 性能监控指标
| 指标名称 | 目标值 | 监控频率 | 告警阈值 |
|----------|--------|----------|----------|
| 技能加载时间 | <100ms | 实时 | >200ms |
| 技能执行时间 | <1s | 实时 | >3s |
| 缓存命中率 | >80% | 每分钟 | <60% |
| 并发执行数 | <50 | 每秒 | >80 |

## 6. 安全考虑

### 6.1 技能隔离机制
每个技能在独立的沙箱环境中执行，实现多层隔离：

1. **进程隔离**：每个技能运行在独立的Node.js进程中
2. **文件系统隔离**：技能只能访问授权目录
3. **网络隔离**：默认禁止外部网络访问，白名单控制
4. **资源限制**：CPU、内存、执行时间硬性限制

### 6.2 权限控制模型
基于HexaCore工具策略模型，实现细粒度的权限控制：

```typescript
interface SkillPermission {
  skillName: string;
  allowedAgents: string[]; // 授权Agent列表
  tools: string[];         // 可用工具列表
  environment: string[];   // 可访问环境变量
  resourceLimits: {
    maxMemory: string;     // 内存限制（如"512m"）
    maxCpu: number;        // CPU配额（0.0-1.0）
    timeout: number;       // 执行超时（毫秒）
  };
}
```

### 6.3 输入验证与过滤
所有输入参数必须经过严格验证，防止注入攻击：

1. **类型验证**：确保参数类型与定义一致
2. **范围验证**：数值参数必须在定义范围内
3. **内容过滤**：字符串参数过滤危险字符
4. **长度限制**：防止缓冲区溢出攻击

### 6.4 审计日志
完整的技能执行审计日志，满足合规要求：

| 日志字段 | 记录内容 | 保留期限 |
|----------|----------|----------|
| 执行时间戳 | 技能开始执行时间 | 1年 |
| 调用方标识 | Agent ID、用户ID | 1年 |
| 技能名称 | 执行的技能标识 | 1年 |
| 输入参数 | 参数哈希值（脱敏） | 90天 |
| 执行结果 | 成功/失败、错误信息 | 90天 |
| 资源消耗 | CPU时间、内存峰值 | 30天 |

#### 8.1.2 核心实现代码
```typescript
// src/skills/weather-query/index.ts
import { SkillBase, SkillInput, SkillOutput } from '@HexaCore/skill-framework';
import axios from 'axios';

interface WeatherParams {
  city: string;
  days?: number;
}

interface WeatherData {
  city: string;
  current: {
    temp: number;
    feels_like: number;
    humidity: number;
    pressure: number;
    wind_speed: number;
    description: string;
    icon: string;
  };
  forecast: Array<{
    date: string;
    temp_min: number;
    temp_max: number;
    description: string;
    precipitation: number;
  }>;
}

export class WeatherQuerySkill extends SkillBase {
  private apiKey: string;
  private baseUrl = 'https://api.openweathermap.org/data/2.5';
  
  constructor() {
    super('weather-query', '1.0.0');
    this.apiKey = process.env.WEATHER_API_KEY || '';
    
    if (!this.apiKey) {
      throw new Error('WEATHER_API_KEY environment variable is required');
    }
  }
  
  async execute(input: SkillInput<WeatherParams>): Promise<SkillOutput<WeatherData>> {
    try {
      const { city, days = 1 } = input.parameters;
      
      // 1. 获取城市坐标
      const geoResponse = await axios.get(`${this.baseUrl}/weather`, {
        params: {
          q: city,
          appid: this.apiKey,
          units: 'metric',
          lang: 'zh_cn'
        }
      });
      
      const { lat, lon } = geoResponse.data.coord;
      
      // 2. 获取当前天气
      const currentWeather = await axios.get(`${this.baseUrl}/weather`, {
        params: {
          lat,
          lon,
          appid: this.apiKey,
          units: 'metric',
          lang: 'zh_cn'
        }
      });
      
      // 3. 获取天气预报
      const forecastResponse = await axios.get(`${this.baseUrl}/forecast/daily`, {
        params: {
          lat,
          lon,
          cnt: Math.min(days, 7),
          appid: this.apiKey,
          units: 'metric',
          lang: 'zh_cn'
        }
      });
      
      // 4. 构建响应数据
      const result: WeatherData = {
        city: currentWeather.data.name,
        current: {
          temp: currentWeather.data.main.temp,
          feels_like: currentWeather.data.main.feels_like,
          humidity: currentWeather.data.main.humidity,
          pressure: currentWeather.data.main.pressure,
          wind_speed: currentWeather.data.wind.speed,
          description: currentWeather.data.weather[0].description,
          icon: `https://openweathermap.org/img/wn/${currentWeather.data.weather[0].icon}@2x.png`
        },
        forecast: forecastResponse.data.list.map((day: any) => ({
          date: new Date(day.dt * 1000).toISOString().split('T')[0],
          temp_min: day.temp.min,
          temp_max: day.temp.max,
          description: day.weather[0].description,
          precipitation: day.pop * 100 // 降水概率百分比
        }))
      };
      
      return {
        success: true,
        data: result,
        metadata: {
          source: 'OpenWeatherMap',
          timestamp: new Date().toISOString(),
          cacheable: true,
          ttl: 1800 // 30分钟缓存
        }
      };
    } catch (error) {
      return {
        success: false,
        error: this.handleError(error),
        metadata: {
          timestamp: new Date().toISOString()
        }
      };
    }
  }
  
  private handleError(error: any): string {
    if (axios.isAxiosError(error)) {
      if (error.response?.status === 401) {
        return '天气API密钥无效，请检查WEATHER_API_KEY配置';
      }
      if (error.response?.status === 404) {
        return '城市名称无效，请检查城市名称拼写';
      }
      return `天气服务暂时不可用: ${error.message}`;
    }
    return `天气查询失败: ${error.message}`;
  }
}

// 导出工厂函数供框架调用
export default function createSkill() {
  return new WeatherQuerySkill();
}
```

```markdown
提供数据导入、清洗、分析、可视化一站式服务，支持多种数据格式和分析方法。

## 数据格式支持
- CSV（逗号分隔、分号分隔）
- JSON（数组、对象嵌套）
- Excel（.xlsx, .xls）
```

#### 8.2.2 核心实现代码
```typescript
// src/skills/data-analysis/index.ts
import { SkillBase, SkillInput, SkillOutput } from '@HexaCore/skill-framework';
import { createClient } from '@redis/client';
import { Pool } from 'pg';
import * as fs from 'fs/promises';
import * as path from 'path';
import { parse } from 'csv-parse';
import { stringify } from 'csv-stringify';

interface AnalysisParams {
  file_url: string;
  analysis_type: 'descriptive' | 'correlation' | 'trend' | 'clustering';
  output_format: 'json' | 'csv' | 'chart' | 'report';
}

interface AnalysisResult {
  summary: {
    row_count: number;
    column_count: number;
    missing_values: number;
    data_types: Record<string, string>;
  };
  visualizations: Array<{
    type: string;
    data: any;
    title: string;
  }>;
  recommendations: string[];
}

export class DataAnalysisSkill extends SkillBase {
  private redis: ReturnType<typeof createClient>;
  private pgPool: Pool;
  
  constructor() {
    super('data-analysis', '1.0.0');
    
    this.redis = createClient({
      url: process.env.REDIS_URL || 'redis://localhost:6379'
    });
    
    this.pgPool = new Pool({
      connectionString: process.env.DATABASE_URL
    });
  }
  
  async initialize(): Promise<void> {
    await this.redis.connect();
  }
  
  async execute(input: SkillInput<AnalysisParams>): Promise<SkillOutput<AnalysisResult>> {
    try {
      const { file_url, analysis_type, output_format } = input.parameters;
      
      // 1. 加载数据
      const data = await this.loadData(file_url);
      
      // 2. 数据清洗
      const cleanedData = await this.cleanData(data);
      
      // 3. 分析处理
      let analysisResult: any;
      switch (analysis_type) {
        case 'descriptive':
          analysisResult = await this.descriptiveAnalysis(cleanedData);
          break;
        case 'correlation':
          analysisResult = await this.correlationAnalysis(cleanedData);
          break;
        case 'trend':
          analysisResult = await this.trendAnalysis(cleanedData);
          break;
        case 'clustering':
          analysisResult = await this.clusteringAnalysis(cleanedData);
          break;
      }
      
      // 4. 生成可视化
      const visualizations = await this.generateVisualizations(
        cleanedData, 
        analysis_type,
        output_format
      );
      
      // 5. 生成建议
      const recommendations = await this.generateRecommendations(cleanedData, analysisResult);
      
      const result: AnalysisResult = {
        summary: {
          row_count: cleanedData.length,
          column_count: Object.keys(cleanedData[0] || {}).length,
          missing_values: this.countMissingValues(cleanedData),
          data_types: this.analyzeDataTypes(cleanedData)
        },
        visualizations,
        recommendations
      };
      
      // 6. 缓存结果
      const cacheKey = `analysis:${Buffer.from(file_url).toString('base64')}:${analysis_type}`;
      await this.redis.setEx(cacheKey, 3600, JSON.stringify(result));
      
      return {
        success: true,
        data: result,
        metadata: {
          processing_time: Date.now() - input.metadata.timestamp,
          analysis_type,
          output_format,
          cache_key: cacheKey
        }
      };
    } catch (error) {
      return {
        success: false,
        error: this.handleError(error),
        metadata: {
          timestamp: new Date().toISOString()
        }
      };
    }
  }
  
  private async loadData(fileUrl: string): Promise<any[]> {
    // 支持URL和本地文件路径
    if (fileUrl.startsWith('http://') || fileUrl.startsWith('https://')) {
      const response = await fetch(fileUrl);
      const contentType = response.headers.get('content-type');
      
      if (contentType?.includes('csv')) {
        const csvText = await response.text();
        return this.parseCSV(csvText);
      } else if (contentType?.includes('json')) {
        return await response.json();
      } else {
        throw new Error(`不支持的内容类型: ${contentType}`);
      }
    } else {
      // 本地文件
      const ext = path.extname(fileUrl).toLowerCase();
      const content = await fs.readFile(fileUrl, 'utf8');
      
      switch (ext) {
        case '.csv':
          return this.parseCSV(content);
        case '.json':
          return JSON.parse(content);
        default:
          throw new Error(`不支持的文件格式: ${ext}`);
      }
    }
  }
  
  private async parseCSV(csvText: string): Promise<any[]> {
    return new Promise((resolve, reject) => {
      parse(csvText, {
        columns: true,
        skip_empty_lines: true,
        trim: true
      }, (err, records) => {
        if (err) reject(err);
        else resolve(records);
      });
    });
  }
  
  private async descriptiveAnalysis(data: any[]): Promise<any> {
    if (data.length === 0) return {};
    
    const columns = Object.keys(data[0]);
    const result: any = {};
    
    for (const col of columns) {
      const values = data.map(row => row[col]).filter(v => v !== null && v !== '');
      
      if (values.length === 0) {
        result[col] = { count: 0, unique: 0 };
        continue;
      }
      
      // 数值类型分析
      const numericValues = values.filter(v => !isNaN(Number(v))).map(v => Number(v));
      
      if (numericValues.length > 0) {
        const sum = numericValues.reduce((a, b) => a + b, 0);
        const mean = sum / numericValues.length;
        const sorted = numericValues.sort((a, b) => a - b);
        const median = sorted[Math.floor(sorted.length / 2)];
        
        result[col] = {
          count: values.length,
          unique: new Set(values).size,
          numeric: {
            count: numericValues.length,
            mean,
            median,
            min: Math.min(...numericValues),
            max: Math.max(...numericValues),
            sum
          }
        };
      } else {
        // 分类数据
        const valueCounts: Record<string, number> = {};
        values.forEach(v => {
          valueCounts[v] = (valueCounts[v] || 0) + 1;
        });
        
        result[col] = {
          count: values.length,
          unique: Object.keys(valueCounts).length,
          top_values: Object.entries(valueCounts)
            .sort((a, b) => b[1] - a[1])
            .slice(0, 5)
        };
      }
    }
    
    return result;
  }
  
  private async generateVisualizations(
    data: any[],
    analysisType: string,
    outputFormat: string
  ): Promise<Array<{ type: string; data: any; title: string }>> {
    const visualizations = [];
    
    // 根据分析类型生成不同的可视化
    if (analysisType === 'descriptive') {
      // 数值分布直方图
      const numericColumns = Object.keys(data[0] || {}).filter(col => {
        const sampleValue = data[0][col];
        return !isNaN(Number(sampleValue)) && sampleValue !== '';
      });
      
      for (const col of numericColumns.slice(0, 3)) {
        const values = data.map(row => Number(row[col])).filter(v => !isNaN(v));
        
        if (values.length > 0) {
          visualizations.push({
            type: 'histogram',
            data: {
              column: col,
              values,
              bin_count: 10
            },
            title: `${col} 数值分布`
          });
        }
      }
    }
    
    return visualizations;
  }
  
  private async generateRecommendations(data: any[], analysisResult: any): Promise<string[]> {
    const recommendations = [];
    
    // 数据质量建议
    const missingValues = this.countMissingValues(data);
    if (missingValues > data.length * 0.1) {
      recommendations.push('数据缺失值较多，建议进行数据清洗或补充');
    }
    
    // 分析结果建议
    if (analysisResult.numeric?.std_dev) {
      const stdDev = analysisResult.numeric.std_dev;
      const mean = analysisResult.numeric.mean;
      if (stdDev > mean * 0.5) {
        recommendations.push('数据标准差较大，可能存在异常值，建议检查数据分布');
      }
    }
    
    return recommendations;
  }
  
  private countMissingValues(data: any[]): number {
    return data.reduce((count, row) => {
      return count + Object.values(row).filter(v => v === null || v === '').length;
    }, 0);
  }
  
  private analyzeDataTypes(data: any[]): Record<string, string> {
    if (data.length === 0) return {};
    
    const firstRow = data[0];
    const result: Record<string, string> = {};
    
    for (const [key, value] of Object.entries(firstRow)) {
      if (typeof value === 'number' || (!isNaN(Number(value)) && value !== '')) {
        result[key] = 'numeric';
      } else if (typeof value === 'string') {
        result[key] = 'string';
      } else if (typeof value === 'boolean') {
        result[key] = 'boolean';
      } else {
        result[key] = 'unknown';
      }
    }
    
    return result;
  }
  
  private handleError(error: any): string {
    return `数据分析失败: ${error.message}`;
  }
}

// 导出工厂函数
export default function createSkill() {
  const skill = new DataAnalysisSkill();
  skill.initialize().catch(console.error);
  return skill;
}
```

```typescript
// src/skills/document-generation/index.ts
import { SkillBase, SkillInput, SkillOutput } from '@HexaCore/skill-framework';
import { OpenAI } from 'openai';
import * as fs from 'fs/promises';
import * as path from 'path';
import * as ejs from 'ejs';
import { marked } from 'marked';
import { chromium } from 'playwright';

interface DocumentParams {
  document_type: 'technical' | 'report' | 'manual' | 'proposal';
  topic: string;
  output_format: 'markdown' | 'html' | 'pdf' | 'word';
  template_name?: string;
}

interface DocumentResult {
  document_url: string;
  metadata: {
    generation_time: number;
    word_count: number;
    template_used: string;
    format: string;
  };
}

export class DocumentGenerationSkill extends SkillBase {
  private openai: OpenAI;
  private templateDir: string;
  
  constructor() {
    super('document-generation', '1.0.0');
    
    this.openai = new OpenAI({
      apiKey: process.env.OPENAI_API_KEY || ''
    });
    
    this.templateDir = process.env.TEMPLATE_DIR || './templates';
  }
  
  async execute(input: SkillInput<DocumentParams>): Promise<SkillOutput<DocumentResult>> {
    const startTime = Date.now();
    
    try {
      const { document_type, topic, output_format, template_name } = input.parameters;
      
      // 1. 选择或加载模板
      const template = await this.loadTemplate(document_type, template_name);
      
      // 2. 生成文档内容
      const content = await this.generateContent(topic, document_type, template);
      
      // 3. 格式转换
      const formattedContent = await this.formatContent(content, output_format);
      
      // 4. 保存文档
      const documentPath = await this.saveDocument(
        topic,
        formattedContent,
        output_format
      );
      
      const result: DocumentResult = {
        document_url: documentPath,
        metadata: {
          generation_time: Date.now() - startTime,
          word_count: this.countWords(content),
          template_used: template_name || `${document_type}_default`,
          format: output_format
        }
      };
      
      return {
        success: true,
        data: result,
        metadata: {
          processing_time: Date.now() - startTime,
          document_type,
          output_format
        }
      };
    } catch (error) {
      return {
        success: false,
        error: this.handleError(error),
        metadata: {
          timestamp: new Date().toISOString(),
          processing_time: Date.now() - startTime
        }
      };
    }
  }
  
  private async loadTemplate(
    documentType: string,
    templateName?: string
  ): Promise<string> {
    const templatePath = templateName 
      ? path.join(this.templateDir, `${templateName}.ejs`)
      : path.join(this.templateDir, `${documentType}_default.ejs`);
    
    try {
      return await fs.readFile(templatePath, 'utf8');
    } catch (error) {
      // 如果模板不存在，使用内置默认模板
      return this.getDefaultTemplate(documentType);
    }
  }
  
  private getDefaultTemplate(documentType: string): string {
    const templates: Record<string, string> = {
      technical: `<h1><%= topic %> 技术文档</h1>
<% sections.forEach(section => { %>
  <h2><%= section.title %></h2>
  <p><%= section.content %></p>
<% }); %>`,
      
      report: `<h1><%= topic %> 分析报告</h1>
<div class="executive-summary">
  <h2>执行摘要</h2>
  <p><%= summary %></p>
</div>
<div class="findings">
  <h2>主要发现</h2>
  <ul>
  <% findings.forEach(finding => { %>
    <li><%= finding %></li>
  <% }); %>
  </ul>
</div>`,
      
      manual: `<h1><%= topic %> 使用手册</h1>
<div class="introduction">
  <h2>简介</h2>
  <p><%= introduction %></p>
</div>
<div class="steps">
  <h2>使用步骤</h2>
  <ol>
  <% steps.forEach(step => { %>
    <li><%= step.description %></li>
  <% }); %>
  </ol>
</div>`,
      
      proposal: `<h1><%= topic %> 项目提案</h1>
<div class="overview">
  <h2>项目概述</h2>
  <p><%= overview %></p>
</div>
<div class="objectives">
  <h2>项目目标</h2>
  <ul>
  <% objectives.forEach(obj => { %>
    <li><%= obj %></li>
  <% }); %>
  </ul>
</div>`
    };
    
    return templates[documentType] || templates.technical;
  }
  
  private async generateContent(
    topic: string,
    documentType: string,
    template: string
  ): Promise<string> {
    // 根据文档类型生成内容
    const templateData = await this.prepareTemplateData(topic, documentType);
    
    // 使用EJS模板引擎渲染
    return ejs.render(template, templateData);
  }
  
  private async prepareTemplateData(
    topic: string,
    documentType: string
  ): Promise<Record<string, any>> {
    const baseData = { topic };
    
    switch (documentType) {
      case 'technical':
        return {
          ...baseData,
          sections: [
            { title: '架构设计', content: '详细描述系统架构和组件交互' },
            { title: '技术选型', content: '说明选择的技术栈和理由' },
            { title: '部署方案', content: '提供完整的部署步骤和配置' }
          ]
        };
        
      case 'report':
        return {
          ...baseData,
          summary: '这是对项目或分析的执行摘要',
          findings: [
            '发现一：系统性能良好',
            '发现二：用户体验有待提升',
            '发现三：安全性符合标准'
          ]
        };
        
      case 'manual':
        return {
          ...baseData,
          introduction: '产品使用手册简介',
          steps: [
            { description: '第一步：安装和配置' },
            { description: '第二步：基本操作' },
            { description: '第三步：高级功能' }
          ]
        };
        
      case 'proposal':
        return {
          ...baseData,
          overview: '项目提案概述',
          objectives: [
            '目标一：提升系统性能',
            '目标二：优化用户体验',
            '目标三：增强安全性'
          ]
        };
        
      default:
        return baseData;
    }
  }
  
  private async formatContent(
    content: string,
    outputFormat: string
  ): Promise<string | Buffer> {
    switch (outputFormat) {
      case 'markdown':
        return content;
        
      case 'html':
        return marked.parse(content);
        
      case 'pdf':
        return await this.generatePDF(content);
        
      case 'word':
        return await this.generateWordDocument(content);
        
      default:
        return content;
    }
  }
  
  private async generatePDF(htmlContent: string): Promise<Buffer> {
    const browser = await chromium.launch();
    const page = await browser.newPage();
    
    await page.setContent(htmlContent);
    const pdfBuffer = await page.pdf({
      format: 'A4',
      printBackground: true,
      margin: { top: '1cm', right: '1cm', bottom: '1cm', left: '1cm' }
    });
    
    await browser.close();
    return pdfBuffer;
  }
  
  private async generateWordDocument(content: string): Promise<Buffer> {
    // 简化处理，实际应使用docx库
    const timestamp = new Date().toISOString().replace(/[:.]/g, '-');
    const docxPath = path.join('/tmp', `document-${timestamp}.docx`);
    
    // 创建简单的docx文件（实际需要更复杂的实现）
    const simpleDocx = this.createSimpleDocx(content);
    
    return Buffer.from(simpleDocx);
  }
  
  private createSimpleDocx(content: string): string {
    // 简化实现，返回一个基本的docx结构
    return `<?xml version="1.0" encoding="UTF-8"?>
<w:document xmlns:w="http://schemas.openxmlformats.org/wordprocessingml/2006/main">
  <w:body>
    <w:p>
      <w:r>
        <w:t>${content}</w:t>
      </w:r>
    </w:p>
  </w:body>
</w:document>`;
  }
  
  private async saveDocument(
    topic: string,
    content: string | Buffer,
    outputFormat: string
  ): Promise<string> {
    const sanitizedTopic = topic.replace(/[^a-zA-Z0-9\u4e00-\u9fa5]/g, '_');
    const timestamp = new Date().toISOString().replace(/[:.]/g, '-');
    const filename = `${sanitizedTopic}_${timestamp}.${outputFormat}`;
    const filepath = path.join('/tmp/documents', filename);
    
    await fs.mkdir(path.dirname(filepath), { recursive: true });
    
    if (Buffer.isBuffer(content)) {
      await fs.writeFile(filepath, content);
    } else {
      await fs.writeFile(filepath, content, 'utf8');
    }
    
    return filepath;
  }
  
  private countWords(content: string): number {
    return content.split(/\s+/).filter(word => word.length > 0).length;
  }
  
  private handleError(error: any): string {
    return `文档生成失败: ${error.message}`;
  }
}

// 导出工厂函数
export default function createSkill() {
  return new DocumentGenerationSkill();
}
```

#### 8.4.2 核心实现代码
```typescript
// src/skills/customer-service/index.ts
import { SkillBase, SkillInput, SkillOutput } from '@HexaCore/skill-framework';
import axios from 'axios';
import { RedisClientType, createClient } from 'redis';

interface CustomerServiceParams {
  message: string;
  session_history?: Array<{
    role: 'user' | 'assistant';
    content: string;
    timestamp: string;
  }>;
}

interface CustomerServiceResult {
  response: string;
  actions: Array<{
    type: 'knowledge_search' | 'ticket_create' | 'escalate' | 'follow_up';
    data: any;
    priority: 'low' | 'medium' | 'high' | 'urgent';
  }>;
  metadata: {
    intent: string;
    confidence: number;
    processing_time: number;
    session_id?: string;
  };
}

export class CustomerServiceSkill extends SkillBase {
  private redis: RedisClientType;
  private knowledgeBaseUrl: string;
  private ticketSystemApiKey: string;
  
  constructor() {
    super('customer-service', '1.0.0');
    
    this.redis = createClient({
      url: process.env.REDIS_URL || 'redis://localhost:6379'
    });
    
    this.knowledgeBaseUrl = process.env.KNOWLEDGE_BASE_URL || 'http://localhost:8000/api/knowledge';
    this.ticketSystemApiKey = process.env.TICKET_SYSTEM_API_KEY || '';
  }
  
  async initialize(): Promise<void> {
    await this.redis.connect();
  }
  
  async execute(input: SkillInput<CustomerServiceParams>): Promise<SkillOutput<CustomerServiceResult>> {
    const startTime = Date.now();
    
    try {
      const { message, session_history = [] } = input.parameters;
      
      // 1. 意图识别
      const intentResult = await this.identifyIntent(message, session_history);
      
      // 2. 根据意图处理
      let response: string;
      let actions: any[] = [];
      
      switch (intentResult.intent) {
        case 'faq_query':
          const faqResponse = await this.handleFaqQuery(message);
          response = faqResponse.answer;
          if (faqResponse.related_questions.length > 0) {
            actions.push({
              type: 'knowledge_search',
              data: faqResponse.related_questions,
              priority: 'low'
            });
          }
          break;
          
        case 'ticket_create':
          const ticketResult = await this.handleTicketCreation(message, session_history);
          response = ticketResult.message;
          actions.push({
            type: 'ticket_create',
            data: ticketResult.ticket_info,
            priority: 'high'
          });
          break;
          
        case 'ticket_status':
          const statusResult = await this.handleTicketStatus(message);
          response = statusResult.message;
          break;
          
        case 'escalation_request':
          const escalationResult = await this.handleEscalationRequest(message, session_history);
          response = escalationResult.message;
          actions.push({
            type: 'escalate',
            data: escalationResult.escalation_info,
            priority: 'urgent'
          });
          break;
          
        default:
          response = await this.handleGeneralQuery(message);
      }
      
      // 3. 会话管理
      const sessionId = await this.manageSession(
        message,
        response,
        intentResult.intent,
        session_history
      );
      
      // 4. 生成响应
      const result: CustomerServiceResult = {
        response,
        actions,
        metadata: {
          intent: intentResult.intent,
          confidence: intentResult.confidence,
          processing_time: Date.now() - startTime,
          session_id: sessionId
        }
      };
      
      // 5. 缓存会话
      if (sessionId) {
        await this.cacheSession(sessionId, result);
      }
      
      return {
        success: true,
        data: result,
        metadata: {
          processing_time: Date.now() - startTime,
          intent: intentResult.intent,
          confidence: intentResult.confidence
        }
      };
    } catch (error) {
      return {
        success: false,
        error: this.handleError(error),
        metadata: {
          timestamp: new Date().toISOString(),
          processing_time: Date.now() - startTime
        }
      };
    }
  }
  
  private async identifyIntent(
    message: string,
    sessionHistory: any[]
  ): Promise<{ intent: string; confidence: number }> {
    // 使用规则引擎和AI模型结合进行意图识别
    
    // 规则匹配
    const ruleIntents = this.matchRules(message);
    if (ruleIntents.length > 0) {
      return {
        intent: ruleIntents[0].intent,
        confidence: ruleIntents[0].confidence
      };
    }
    
    // AI模型识别
    const aiIntents = await this.classifyWithAI(message, sessionHistory);
    return {
      intent: aiIntents.intent,
      confidence: aiIntents.confidence
    };
  }
  
  private matchRules(message: string): Array<{ intent: string; confidence: number }> {
    const rules = [
      {
        keywords: ['问题', '故障', '错误', 'bug', 'issue'],
        intent: 'ticket_create',
        confidence: 0.9
      },
      {
        keywords: ['状态', '进度', '查询', '查看', '跟踪'],
        intent: 'ticket_status',
        confidence: 0.8
      },
      {
        keywords: ['怎么', '如何', '怎样', '步骤', '教程'],
        intent: 'faq_query',
        confidence: 0.85
      },
      {
        keywords: ['投诉', '举报', '紧急', '加急', '马上'],
        intent: 'escalation_request',
        confidence: 0.95
      }
    ];
    
    const lowerMessage = message.toLowerCase();
    const matchedRules = rules.filter(rule =>
      rule.keywords.some(keyword => lowerMessage.includes(keyword))
    );
    
    return matchedRules.map(rule => ({
      intent: rule.intent,
      confidence: rule.confidence
    }));
  }
  
  private async classifyWithAI(
    message: string,
    sessionHistory: any[]
  ): Promise<{ intent: string; confidence: number }> {
    // 简化处理，实际应调用AI服务
    // 这里返回默认意图
    return {
      intent: 'general_query',
      confidence: 0.7
    };
  }
  
  private async handleFaqQuery(message: string): Promise<{
    answer: string;
    related_questions: string[];
  }> {
    // 查询知识库
    const knowledgeResult = await this.searchKnowledgeBase(message);
    
    if (knowledgeResult.answers.length > 0) {
      return {
        answer: knowledgeResult.answers[0].content,
        related_questions: knowledgeResult.related_questions
      };
    }
    
    // 默认回答
    return {
      answer: '感谢您的提问。我已经记录了您的问题，稍后会有专员为您详细解答。',
      related_questions: []
    };
  }
  
  private async searchKnowledgeBase(query: string): Promise<{
    answers: Array<{ id: string; content: string; score: number }>;
    related_questions: string[];
  }> {
    try {
      const response = await axios.post(
        `${this.knowledgeBaseUrl}/search`,
        { query, top_k: 5 },
        {
          headers: { 'Content-Type': 'application/json' }
        }
      );
      
      return response.data;
    } catch (error) {
      console.error('知识库查询失败:', error);
      return { answers: [], related_questions: [] };
    }
  }
  
  private async handleTicketCreation(
    message: string,
    sessionHistory: any[]
  ): Promise<{
    message: string;
    ticket_info: {
      id: string;
      status: string;
      priority: string;
      created_at: string;
    };
  }> {
    // 提取问题信息
    const problemType = this.extractProblemType(message);
    const urgencyLevel = this.assessUrgencyLevel(message, sessionHistory);
    
    // 创建工单
    const ticketId = `TICKET-${Date.now()}-${Math.random().toString(36).substr(2, 9)}`;
    
    // 简化处理，实际应调用工单系统API
    return {
      message: `您的工单已成功创建，工单号为：${ticketId}。我们的技术支持团队将在24小时内处理您的问题。`,
      ticket_info: {
        id: ticketId,
        status: 'open',
        priority: urgencyLevel,
        created_at: new Date().toISOString()
      }
    };
  }
  
  private extractProblemType(message: string): string {
    // 简化处理，实际应使用NLP技术
    if (message.includes('登录')) return 'login_issue';
    if (message.includes('支付')) return 'payment_issue';
    if (message.includes('订单')) return 'order_issue';
    return 'general_issue';
  }
  
  private assessUrgencyLevel(
    message: string,
    sessionHistory: any[]
  ): 'low' | 'medium' | 'high' | 'urgent' {
    // 根据消息内容和历史会话评估紧急程度
    if (message.includes('紧急') || message.includes('马上')) return 'urgent';
    if (sessionHistory.some(h => h.content.includes('问题'))) return 'high';
    return 'medium';
  }
  
  private async handleTicketStatus(message: string): Promise<{ message: string }> {
    // 提取工单号
    const ticketId = this.extractTicketId(message);
    
    if (!ticketId) {
      return {
        message: '请提供您的工单号，以便查询处理进度。'
      };
    }
    
    // 简化处理，实际应查询工单系统
    return {
      message: `工单 ${ticketId} 当前状态为：处理中，预计将在2小时内完成。`
    };
  }
  
  private extractTicketId(message: string): string | null {
    // 简化处理，实际应使用正则表达式
    const match = message.match(/TICKET-\w+/i);
    return match ? match[0] : null;
  }
  
  private async handleEscalationRequest(
    message: string,
    sessionHistory: any[]
  ): Promise<{
    message: string;
    escalation_info: {
      level: string;
      reason: string;
      timestamp: string;
    };
  }> {
    // 升级处理
    const escalationLevel = this.determineEscalationLevel(message, sessionHistory);
    const reason = this.extractEscalationReason(message);
    
    return {
      message: `您的问题已经升级到${escalationLevel}级别处理，将有专员在30分钟内联系您。`,
      escalation_info: {
        level: escalationLevel,
        reason,
        timestamp: new Date().toISOString()
      }
    };
  }
  
  private determineEscalationLevel(
    message: string,
    sessionHistory: any[]
  ): string {
    // 根据情况确定升级级别
    if (sessionHistory.some(h => h.content.includes('投诉'))) return 'L2';
    if (message.includes('严重')) return 'L1';
    return 'L3';
  }
  
  private extractEscalationReason(message: string): string {
    // 简化处理
    if (message.includes('时间长')) return '处理时间过长';
    if (message.includes('态度')) return '服务态度问题';
    return '客户要求升级';
  }
  
  private async handleGeneralQuery(message: string): Promise<string> {
    // 通用应答
    return '感谢您的咨询。我已经记录了您的问题，稍后会有专员为您详细解答。如果您需要立即帮助，请致电我们的客服热线：400-123-4567。';
  }
  
  private async manageSession(
    message: string,
    response: string,
    intent: string,
    sessionHistory: any[]
  ): Promise<string | null> {
    // 生成会话ID
    const sessionId = `session_${Date.now()}_${Math.random().toString(36).substr(2, 9)}`;
    
    // 存储会话历史
    const updatedHistory = [
      ...sessionHistory,
      {
        role: 'user' as const,
        content: message,
        timestamp: new Date().toISOString()
      },
      {
        role: 'assistant' as const,
        content: response,
        timestamp: new Date().toISOString()
      }
    ];
    
    // 限制历史记录长度
    const maxHistoryLength = 20;
    const trimmedHistory = updatedHistory.slice(-maxHistoryLength);
    
    // 存储到Redis（简化处理）
    await this.redis.setEx(
      `session:${sessionId}`,
      86400, // 24小时
      JSON.stringify({
        history: trimmedHistory,
        last_activity: new Date().toISOString(),
        intent
      })
    );
    
    return sessionId;
  }
  
  private async cacheSession(sessionId: string, result: CustomerServiceResult): Promise<void> {
    // 缓存最近会话
    const cacheKey = `recent_session:${sessionId}`;
    await this.redis.setEx(
      cacheKey,
      3600, // 1小时
      JSON.stringify(result)
    );
  }
  
  private handleError(error: any): string {
    if (axios.isAxiosError(error)) {
      if (error.response?.status === 401) {
        return '知识库API密钥无效，请检查KNOWLEDGE_BASE_URL和API密钥配置';
      }
      return `客服服务暂时不可用: ${error.message}`;
    }
    return `客服应答失败: ${error.message}`;
  }
}
```
